「CodeForces 1234F」Yet Another Substring Reverse

2019-10-11 #状压DP

You are given a string 𝑠 consisting only of first 20 lowercase Latin letters ('a', 'b', ..., 't').

Recall that the substring 𝑠[𝑙;𝑟] of the string 𝑠 is the string 𝑠𝑙𝑠𝑙+1…𝑠𝑟. For example, the substrings of "codeforces" are "code", "force", "f", "for", but not "coder" and "top".

You can perform the following operation no more than once: choose some substring 𝑠[𝑙;𝑟] and reverse it (i.e. the string 𝑠𝑙𝑠𝑙+1…𝑠𝑟 becomes 𝑠𝑟𝑠𝑟−1…𝑠𝑙).

Your goal is to maximize the length of the maximum substring of 𝑠 consisting of distinct (i.e. unique) characters.

The string consists of distinct characters if no character in this string appears more than once. For example, strings "abcde", "arctg" and "minecraft" consist of distinct characters but strings "codeforces", "abacaba" do not consist of distinct characters.

「CodeForces 1234F」Yet Another Substring Reverse

题解

反转等价于找段字符串，使得他们之间没有字符相同，并且长度只和最长
状压状压，DP[i] 表示选取二进制位选取与不选取的最长长度，如果选取了某段，那么与它不同的另一段等于 DP[((1<<m)-1)^1],枚举统计最大情况

代码

#include <cstdio>
#include <cstring>
#include <algorithm>
using namespace std;
const int maxd = 1e6+10;
char s[maxd];
int dp[1<<21],vis[21];
int main()
{
	// freopen("a.in","r",stdin);
	// freopen("k.out","w",stdout);
	scanf("%s",s);
	for(int i=0;s[i];i++)
	{
		int now = 0 , cnt = 0;
		memset(vis,0,sizeof(vis));
		for(int j=i;s[j];j++)
		{
			int x = s[j]-'a';
			if(vis[x]) break;
			vis[x] =1; now |= (1<<x) ;
			cnt++;
			dp[now] = cnt;
		}
	}
	for(int i=0;i<(1<<20);i++)
		for(int j=0;j<20;j++)
			if(i & (1<<j)) 
				dp[i] = max(dp[i],dp[i^(1<<j)]); // 选取子集中最大方案数量
	int ans = 0;
	for(int i=0;i<(1<<20);i++)
		ans = max(ans,dp[i] + dp[i ^ ((1<<20) - 1)]);
	printf("%d",ans);
	return 0;
}
