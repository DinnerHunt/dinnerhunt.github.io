「HDU 6184」Counting Stars

2019-12-01 三元环 组合数学

Little A is an astronomy lover, and he has found that the sky was so beautiful!

So he is counting stars now!

There are n stars in the sky, and little A has connected them by m non-directional edges.

It is guranteed that no edges connect one star with itself, and every two edges connect different pairs of stars.

Now little A wants to know that how many different "A-Structure"s are there in the sky, can you help him?

An "A-structure" can be seen as a non-directional subgraph G, with a set of four nodes V and a set of five edges E.

If V=(A,B,C,D) and E=(AB,BC,CD,DA,AC), we call G as an "A-structure".

It is defined that "A-structure" G1=V1+E1 and G2=V2+E2 are same only in the condition that V1=V2 and E1=E2.

题解

给定的图可以看成是两个共单边的三元环
所以题目转化为求三元环的方案数
统计每条边被多少三元环覆盖，计为cnt[i]cnt[i]cnt[i]
通过组合排列，可得到答案为

ans=∑inCcnt[i]2ans = \sum_i^n C_{cnt[i]}^2 ans=i∑n​Ccnt[i]2​

代码

#include <cstdio>
#include <vector>
#include <cstring>
#include <algorithm>
using namespace std;
const int maxd = 2e5+10;
int d[maxd],vis[maxd],cnt[maxd],n,m;
typedef pair<int,int> P;
vector<P> G[maxd],e;
int main()
{
//    freopen("a.in","r",stdin);
//    freopen("k.out","w",stdout);
    while(scanf("%d %d",&n,&m)!=EOF)
    {
        for (int i = 1; i <= m; i++)
        {
            int u,v;scanf("%d %d", &u, &v);
            e.push_back(P(u,v));
            ++d[u], ++d[v];
        }
        for (int i = 0; i < m; i++)
        {
            int u = e[i].first, v = e[i].second;
            if (d[u] < d[v] || (d[u] == d[v] && u > v)) swap(u, v);

            G[u].push_back(P(v, i+1));
        }
        for (int i = 1; i <= n; i++)
        {
            for (P u:G[i]) vis[u.first] = u.second;
            for (P u:G[i])
                for (P v:G[u.first])
                    if (vis[v.first])
                        ++cnt[v.second], ++cnt[u.second], ++cnt[vis[v.first]];

            for (P u:G[i]) vis[u.first] = 0;
        }
        long long ans = 0ll;
        for (int i = 1; i <= m; i++)
        {
            ans += 1ll * cnt[i] * (cnt[i] - 1) / 2;
            cnt[i] = 0;
        }

        for(int i=1;i<=n;i++) G[i].clear(),d[i] = 0; e.clear();
        printf("%lld\n", ans);
    }
    return 0;
}
