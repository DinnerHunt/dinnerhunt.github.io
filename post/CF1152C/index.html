CodeForces-1152C Neko does Maths

2019-05-05 数论

Neko loves divisors. During the latest number theory lesson, he got an interesting exercise from his math teacher.

Neko has two integers 𝑎 and 𝑏. His goal is to find a non-negative integer 𝑘 such that the least common multiple of 𝑎+𝑘 and 𝑏+𝑘 is the smallest possible. If there are multiple optimal integers 𝑘, he needs to choose the smallest one.

Given his mathematical talent, Neko had no trouble getting Wrong Answer on this problem. Can you help him solve it?

[CodeForces 1152C]Neko does Maths

题解

原式等于

(a+k)⋅(b+k)gcd(a+k,b+k) \frac{(a+k) \cdot (b+k)}{ gcd(a+k,b+k) } gcd(a+k,b+k)(a+k)⋅(b+k)​

设

d=gcd(a+k,b+k) d = gcd(a+k,b+k) d=gcd(a+k,b+k)

a+k=x1∗d a+k = x1 * d a+k=x1∗d

b+k=x2∗d b+k = x2 * d b+k=x2∗d

联立方程组

(x1−x2)⋅d=a−b(x1-x2) \cdot d = a-b (x1−x2)⋅d=a−b

(a−b)=0(mod p)(a - b) = 0 (mod \ p) (a−b)=0(mod p)

a+k=b+k=0(mod p)a + k = b + k = 0 (mod \ p) a+k=b+k=0(mod p)

由此得出 d 为 a-b 的因数，并且 d 为 a+k , b+k 的因数

k=(d−a%d)%dk = (d - a \% d) \% d k=(d−a%d)%d

代码

#include <cstdio>
#include <iostream>
#include <algorithm>
using namespace std;
typedef long long ll;
ll mx = 1ll * 1e9 * 1e9,ans = 0;
ll a,b,c;
ll gcd(ll x,ll y)
{
    if(!y) return x;
    return gcd(y,x%y);
}
void solve(ll d)
{
    if(a%d != b%d) return;
    ll k = (d - a%d) %d,lcm = (a+k)*(b+k)/d;
    if(lcm <= mx)
    {
        if(lcm == mx) ans= min(ans,k);
        else ans = k;
        mx = lcm;
    }
}
int main()
{
    // freopen("a.in","r",stdin);
    // freopen("k.out","w",stdout);
    cin >> a >> b;
    c = abs(a-b);
    for(ll i=1ll;i*i<=c;i++)
    {
        if(c%i == 0)
        {
            solve(i);
            if(i*i!=c) solve(c/i);
        }
    }
    cout << ans;
    return 0;
}
