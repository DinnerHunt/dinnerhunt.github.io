CodeForces-402E Strictly Positive Matrix

2019-05-05 连通图

You have matrix a of size n × n. Let's number the rows of the matrix from 1 to n from top to bottom, let's number the columns from 1 to n from left to right. Let's use aija_{ij}aij​ to represent the element on the intersection of the i-th row and the j-th column.

Matrix a meets the following two conditions:

for any numbers i, j (1 ≤ i, j ≤ n) the following inequality holds: aij ≥ 0a_{ij} ≥ 0aij​ ≥ 0;
∑i=1naii>0\sum_{i=1}^n a_{ii}>0∑i=1n​aii​>0

Matrix b is strictly positive, if for any numbers i, j (1 ≤ i, j ≤ n) the inequality bijb_{ij}bij​ > 0 holds. You task is to determine if there is such integer k ≥ 1, that matrix aka^kak is strictly positive.

题解

将矩阵视为一个点数为 n 的矩阵
如果 arr[i][j] > 0 则说明 i,j 之间存在一条有向边
如果图是一个强联通图，说明满足，否则不满足

代码

#include <cstdio>
#include <algorithm>
#include <cstring>
using namespace std;
const int MAX_N = 2005;
const int MAX_M = 2005*2005;
struct edge {
    int v, next;
} e[MAX_M];
int p[MAX_N], eid;
void init(){
  memset(p,-1,sizeof(p));
  eid=0;
}
void insert(int a,int b){
    e[eid].v = b;
    e[eid].next = p[a];
    p[a] = eid++;
}
int dfn[MAX_N], low[MAX_N];  
int idx = 0;
int belong[MAX_N], scc = 0; 
int s[MAX_N], top = 0,x,n; 
bool in_stack[MAX_N];
void tarjan(int u) {
        dfn[u] = low[u] = ++idx;
        s[top++] = u;
        in_stack[u] = true;
        for (int i = p[u]; i != -1; i = e[i].next) 
        {
            int v = e[i].v;
            if (!dfn[v]) {
                tarjan(v);
                low[u] = min(low[u], low[v]);
            } else if (in_stack[v]) {
                low[u] = min(low[u], dfn[v]);
            }
        }
        if (dfn[u] == low[u]) {
            ++scc;
            do {
                belong[s[--top]] = scc;
                in_stack[s[top]] = false;
            } while (s[top] != u);
        }
}
int main()
{
    init();
    // freopen("a.in","r",stdin);
    // freopen("k.out","w",stdout);
    scanf("%d",&n);
    for(int i=1;i<=n;i++) 
        for(int j=1;j<=n;j++)
        {
            scanf("%d",&x);
            if(x) insert(i,j);
        }
    for(int i=1;i<=n;i++) if(!dfn[i]) tarjan(i);
    if(scc==1) printf("YES");
    else printf("NO");
    return 0;
}
