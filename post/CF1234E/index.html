「CodeForces 1234E」Special Permutations

2019-10-12 #枚举 #树状数组

Let's define 𝑝𝑖(𝑛) as the following permutation: [𝑖,1,2,…,𝑖−1,𝑖+1,…,𝑛]. This means that the 𝑖-th permutation is almost identity (i.e. which maps every element to itself) permutation but the element 𝑖 is on the first position. Examples:

𝑝1(4)=[1,2,3,4];
𝑝2(4)=[2,1,3,4];
𝑝3(4)=[3,1,2,4];
𝑝4(4)=[4,1,2,3].
You are given an array 𝑥1,𝑥2,…,𝑥𝑚 (1≤𝑥𝑖≤𝑛).

Let 𝑝𝑜𝑠(𝑝,𝑣𝑎𝑙) be the position of the element 𝑣𝑎𝑙 in 𝑝. So, 𝑝𝑜𝑠(𝑝1(4),3)=3,𝑝𝑜𝑠(𝑝2(4),2)=1,𝑝𝑜𝑠(𝑝4(4),4)=1.

Let's define a function 𝑓(𝑝)=∑𝑖=1𝑚−1|𝑝𝑜𝑠(𝑝,𝑥𝑖)−𝑝𝑜𝑠(𝑝,𝑥𝑖+1)|, where |𝑣𝑎𝑙| is the absolute value of 𝑣𝑎𝑙. This function means the sum of distances between adjacent elements of 𝑥 in 𝑝.

Your task is to calculate 𝑓(𝑝1(𝑛)),𝑓(𝑝2(𝑛)),…,𝑓(𝑝𝑛(𝑛)).

题解

两部分
第一部分是P(i) 时，第i个数值的修改，会印象到pos-1与pos+1的点，这里修改时间复制度是O(n)的
第二部分是小于i的值都会+1，如果当前修改位i,如果存在两个相邻的值为x,y，切 x < i < y 或 y < i < x ,那么就会存在贡献减少的情况，这里可以用树状数组统计下，然后每次枚举点的时候删除即可

代码

#include <cstdio>
#include <cstring>
#include <vector>
#include <algorithm>
using namespace std;
const int maxd = 2e5+10;
vector<int> vis[maxd];
int n,m,a[maxd];
int abs(int x)
{
	if(x < 0) return -x;
	return x;
}
int t[maxd<<1];
void add(int x,int v)
{
	for(;x<maxd*2;x+=(x&-x)) t[x]+=v;
}
int query(int x)
{
	int sum = 0;
	for(;x;x-=(x&-x)) sum += t[x];
	return sum;
}
int main()
{
	// freopen("a.in","r",stdin);
	// freopen("k.out","w",stdout);
	scanf("%d %d",&n,&m);
	for(int i=1;i<=m;i++) 
	{
		scanf("%d",&a[i]);
		vis[a[i]].push_back(i);
	}
	long long ans = 0ll;
	for(int i=2;i<=m;i++) 
	{
		ans += abs(a[i] - a[i-1]);
		int x = a[i], y = a[i-1];
		if(x > y) swap(x,y);
		if(y - x > 1)
		{
			add(x+1,1);
			add(y,-1);
		}
	}
	//printf("%lld ",ans);
	for(int i=1;i<=n;i++)
	{
		long long sum = ans;
		sum -= query(i);
		//printf("%d %lld\n",i,sum);
		for(int j=0;j<vis[i].size();j++)
		{
			int pos = vis[i][j];
			int x = a[pos-1],y = a[pos+1];
			//printf("	%d %d %d\n",pos,x,y);
			if(x != i && x) 
			{
				sum -= abs(i - x);
				if(x<i) sum += x;
				else sum+= 	x-1;
			}
			if(y != i && y)
			{
				sum -= abs(i-y);
				if(y<i) sum+=y;
				else sum+= y-1;
			}
		}
		printf("%lld\n",sum);
	}
 
	return 0;
}
