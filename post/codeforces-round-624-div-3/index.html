Codeforces Round #624 (Div. 3)

2020-02-28

Code

A - Add Odd or Subtract Even
B - WeirdSort
C - Perform the Combo
D - Three Integers
E - Construct the Binary Tree
F - Moving Points

A

很显然，最坏情况只需两步就能使他们相等，另外情况分别考虑一下即可


B

不能动的位置相当于截断成mmm个区间，然后区间内排序，最后判断下是否满足答案即可


C

前缀和记录一下字符串字符出现的次数即可


D

随便选择a,b,ca,b,ca,b,c中的一个数枚举，然后另外两个数的范围就缩小了，然后暴力枚举，统计答案即可，（枚举的范围注意调高点）


E

构造题，先按完全二叉树的形式构造，这是数的最小值形态。
记录数的最左端的链，因为为了维护树的形态，我们让该链不允许挪动，然后枚举其他点，尝试是否能将点往下提


F

因为时间是无限的。因此，考虑两个点a,ba,ba,b，如果ax<bxa_x < b_xax​<bx​但av>bva_v > b_vav​>bv​，那么他们在某一时刻的距离一定是为0的，因此对于每个点aaa我们需要统计其与点bbb距离，当前仅当ax<bx且av<bva_x < b_x \text{且} a_v < b_vax​<bx​且av​<bv​ 即可
这里可以用树状数组维护，维护方式类似与逆序对
