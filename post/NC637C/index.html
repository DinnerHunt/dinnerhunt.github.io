「16届同大邀请赛」 半数素

2019-05-05 #数论

最近张老师对半素数感兴趣。半素数(semi-prime)是可以表示成两个素数乘积的数。比如4和10是半素数，因为4=2×2，10=2×5。而8不是半素数，因为8=2×2×2。他想知道某一个l到r的闭区间内有多少个这样的数。但是这个问题太困难了，他想让你帮他解决。

题目链接🔗

16届 同大邀请赛 半数素

题解

考虑埃氏筛法，将l-r区间的因子一定是在1e5以内，我们枚举1e5以内的数，并考虑他们在区间l-r中数的贡献，并对每个数用数组记录因子个数

如果一个数的因子个数为2，那么他则是半素数

代码
#include <cstdio>
#include <vector>
#include <cstring>
#include <algorithm>
using namespace std;
typedef long long ll;
const int maxd = 1e5+10;
vector<ll> v[maxd];
ll a[maxd];
int main()
{
    // freopen("a.in","r",stdin);
    // freopen("k.out","w",stdout);
    ll l,r;scanf("%lld %lld",&l,&r);
    for(ll i=l;i<=r;i++) a[i-l] = i;
    for(ll i=2;i*i<=r;i++)  //枚举因子
        for(ll j = (i+l-1)/i*i;j<=r;j+=i) //枚举因子的作用区域 n^(1/2)
            while(a[j-l] % i == 0)
            {
                v[j-l].push_back(i);    //添加因子
                a[j-l]/=i;
            }
    ll ans = 0;
    for(ll i=0;i<=r-l;i++)
    {
        if(a[i] > 1) v[i].push_back(a[i]);
        sort(v[i].begin(),v[i].end());
        if(v[i].size() == 2) ans++;
    }
    printf("%lld\n",ans);
    for(ll i=0;i<=r-l;i++)
        if(v[i].size() == 2) printf("%lld %lld %lld\n",i+l,v[i][0],v[i][1]);
    return 0;
}
