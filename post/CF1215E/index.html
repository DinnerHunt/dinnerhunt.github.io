гҖҢCodeForces 1215EгҖҚ Marbles

2019-12-28 зҠ¶еҺӢDP

Monocarp has arranged рқ‘ӣ colored marbles in a row. The color of the рқ‘–-th marble is рқ‘Һрқ‘–. Monocarp likes ordered things, so he wants to rearrange marbles in such a way that all marbles of the same color form a contiguos segment (and there is only one such segment for each color).

In other words, Monocarp wants to rearrange marbles so that, for every color рқ‘—, if the leftmost marble of color рқ‘— is рқ‘ҷ-th in the row, and the rightmost marble of this color has position рқ‘ҹ in the row, then every marble from рқ‘ҷ to рқ‘ҹ has color рқ‘—.

To achieve his goal, Monocarp can do the following operation any number of times: choose two neighbouring marbles, and swap them.

You have to calculate the minimum number of operations Monocarp has to perform to rearrange the marbles. Note that the order of segments of marbles having equal color does not matter, it is only required that, for every color, all the marbles of this color form exactly one contiguous segment.

const ll maxd = 4e5+10;
const ll maxn = (1<<20)-1;

ll a[maxd],b[maxd],cnt[27];
ll num[27][27],dp[(1<<20)+10];
int main()
{
//    __IN;__OUT;
    ll n;RLL(n);
    FOR(i,1,n)RLL(a[i]);
    FOR(i,1,n)
    {
        FOR(j, 1, 20)
            num[a[i]][j] += cnt[j];
        cnt[a[i]]++;
    }
    memset(dp,0x3f,sizeof(dp));
    dp[0] = 0;
    FOR(i,1,maxn)
        FOR(j,0,19)
        {
            if(!((i>>j)&1)) continue; //жһҡдёҫз¬¬jдёӘдҪҚзҪ®
            ll t=0;
            FOR(k,0,19)
            {
                if((i>>k)&1) continue;  // дҪҚзҪ®зЎ®е®ҡдәҶпјҢи·іиҝҮ
                t += num[j+1][k+1];
            }
            dp[i] = min(dp[i],dp[i^(1<<j)]+t);
        }
    PLL(dp[maxn]);
    return 0;
}
