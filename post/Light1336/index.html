Sigma function is an interesting function in Number Theory. It is denoted by the Greek letter Sigma (σ). This function actually denotes the sum of all divisors of a number. For example σ(24) = 1+2+3+4+6+8+12+24=60. Sigma of small numbers is easy to find but for large numbers it is very difficult to find in a straight forward way. 题解

因为和为偶数与和为奇数互斥，所以我们可以先求出和为奇数的次数，然后减去即可得到偶数个数

根据因数和的定理，我们知道因数和是由质数因子决定的
因为只有奇数个奇数相乘，才能保证结果是奇数

我们先看单项
因为2的任意项数之和加1都是奇数，所以2的项都是奇数

其他质数都是奇数，所以质数的次数项一定是偶数，才能保证这每项内的数为奇数
质数的每次数项都是偶数的数为完全平方数

还有种可能是2为奇数次，但也满足单项为奇数
2为偶数次项时也为完全平方数，为了防止不重复计算，我们只减去完全平方数和完全平方数乘2即可

1-n中完全平方数的个数为sqrt(n),完全平方数乘

#include <cmath>
