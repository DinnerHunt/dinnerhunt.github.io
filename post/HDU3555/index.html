HDU-3555 Bomb

2019-05-05 数位DP

The counter-terrorists found a time bomb in the dust. But this time the terrorists improve on the time bomb. The number sequence of the time bomb counts from 1 to N. If the current number sequence includes the sub-sequence "49", the power of the blast would add one point. Now the counter-terrorist knows the number N. They want to know the final points of the power. Can you help them?

题目链接🔗

[HDU 3555] Bomb

题解

数位DP,分类讨论 先考虑长度，长度为n时最多会有多少个数包含62 然后分类讨论

代码

#include <cstdio>
#include <algorithm>
using namespace std;
long long n,arr[15],dp[15][3];
void init()
{
    dp[0][0] = 1; // 前i为不包含49的个数
    dp[0][1] = dp[0][2] = 0; // 最高位为9的个数，包含49的个数
    for(int i=1;i<15;i++)
    {
        dp[i][0] = dp[i-1][0] * 10 - dp[i-1][1];
        dp[i][1] = dp[i-1][0];
        dp[i][2] = dp[i-1][1] + dp[i-1][2] * 10;
    }
}
long long solve(long long n)
{
    long long len=0,ans=0,flag = 0;
    while(n) arr[++len]=n%10,n/=10; arr[len+1] = 0;
    for(int i=len;i>0;i--)
    {
        ans += arr[i] * dp[i-1][2];
        if(flag) ans += dp[i-1][0] * arr[i];
        if(!flag&&arr[i]>4) ans += dp[i-1][1];
        if(arr[i+1]==4&&arr[i]==9) flag = 1;
    }
    return ans;
}
int main()
{
    // freopen("a.in","r",stdin);
    // freopen("k.out","w",stdout);
    init();int t;scanf("%d",&t);
    for(int i=1;i<=t;i++)
    {
        scanf("%lld",&n);
        printf("%lld\n",solve(n+1));
    }
    return 0;
}
