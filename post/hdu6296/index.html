「HDU 6296」代码派对

2020-03-19 前缀和 组合数学

比特镇的算法竞赛选手们应邀来到了小Q家中参加盛大的"代码派对"。小Q计划在家中举行一场ACM赛制的联谊赛，3个人为一队进行比赛。

为了让比赛更加有趣，也为了让大家能多多了解他人，小Q在花园的地上画出了一个1000×1000的网格图，从上往下依次编号为第1行到第1000行，从左往右依次编号为第1列到第1000列。小Q让每一名选手选择一个网格图内平行坐标轴的子矩形(x1,y1)−(x2,y2)，表示以(x1,y1)和(x2,y2)为对顶点的矩形，包含所有满足x∈[x1,x2]且y∈[y1,y2]的格子。

小Q规定，i,j,k(1≤i < j < k ≤n)三人组成一支三人队伍能参赛，当且仅当这三个人选择的子矩形包含至少一个公共格子。请写一个程序，帮助小Q统计有多少种三人组合组成的队伍可以参赛。

「HDU 6296」代码派对

题解 ❓

一道非常妙的题目
首先可以做一次二维前缀和，然后对于每个格子i,ji,ji,j可以得到方案数Ca[i][j]3C_{a[i][j]}^3Ca[i][j]3​，计为 S0S_0S0​
但很显然，这样会重复统计，比如两个矩阵交于多个格子
于是我们考虑求出重复的数量
对于每个矩阵
将其最左边一列去掉，进行一次统计方案数，计为S1S_1S1​
将其最上方一列去掉，进行一次统计方案数，计为S2S_2S2​
将其最上方一列和左方一列去掉，再进行一次统计方案数，计为S3S_3S3​
答案则为 S0−S1−S2+S3S_0 - S_1 - S_2 +S_3S0​−S1​−S2​+S3​

代码

const ll N = 1e5+10,M = 1e3+10;
const ll mod = 998244353;
ll C[N],a[N][5],pre[M][M],n;
ll gao(ll x,ll y)
{
    ll ans = 0;MM(pre,0);
    FOR(i,1,n)
    {
        ll x1 = a[i][1] + x ,y1 = a[i][2] + y ,x2 = a[i][3],y2 = a[i][4];
        pre[x1][y1]++; pre[x2+1][y2+1]++;
        pre[x2+1][y1]--;pre[x1][y2+1]--;
    }
    FOR(i,1,M-1) FOR(j,1,M-1) pre[i][j] += pre[i][j-1] + pre[i-1][j] - pre[i-1][j-1] ,ans+=C[pre[i][j]];
//    PLN(ans);
    return ans;
}
int main()
{
//    __IN;__OUT
    FOR(i,3,N-1) C[i] = i*(i-1)/2*(i-2)/3;
    ll T;RLL(T);
    while(T--)
    {
        RLL(n);
        FOR(i,1,n) FOR(j,1,4)  RLL(a[i][j]);
        PLN(gao(0,0)-gao(0,1)-gao(1,0)+gao(1,1));
    }
    return 0;
}
