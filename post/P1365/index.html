「Luogu P1365」 WJMZBMR打osu! / Easy

2019-05-05 期望概率

某一天WJMZBMR在打osu~~~但是他太弱逼了，有些地方完全靠运气:(

我们来简化一下这个游戏的规则

有n次点击要做，成功了就是o，失败了就是x，分数是按combo计算的，连续 a个combo就有a\times a分，combo就是极大的连续o。

比如ooxxxxooooxxx，分数就是 2×2+4×4=4+16=202 \times 2 + 4 \times 4 = 4 +16=202×2+4×4=4+16=20

Sevenkplus闲的慌就看他打了一盘，有些地方跟运气无关要么是o要么是x，有些地方o或者x各有50%的可能性，用?号来表示。

比如oo?xx就是一个可能的输入。 那么WJMZBMR这场osu的期望得分是多少呢？

比如oo?xx的话，?是o的话就是oooxx => 9，是x的话就是ooxxx => 4

期望自然就是(4+9)/2 =6.5 了

题解

设 dp[i]dp[i]dp[i] 为前 iii 个字符的期望得分， len[i]len[i]len[i] 为当前的期望 O 的连续长度 然后我们考虑 3 种情况

X:dp[i]=dp[i−1]X : dp[i] = dp[i-1]X:dp[i]=dp[i−1]

      len[i]=0\ \ \ \ \ \ len[i] = 0      len[i]=0

O:dp[i]=dp[i−1]+2∗len[i−1]+1O : dp[i] = dp[i-1] + 2*len[i-1] + 1O:dp[i]=dp[i−1]+2∗len[i−1]+1

      len[i]=len[i−1]+1\ \ \ \ \ \ len[i] = len[i-1] + 1      len[i]=len[i−1]+1

?:dp[i]=dp[i−1]+len[i−1]+0.5? : dp[i] = dp[i-1] + len[i-1] + 0.5?:dp[i]=dp[i−1]+len[i−1]+0.5

      len[i]=len[i−1]+12\ \ \ \ \ \ len[i] = \frac{len[i-1]+1}{2}      len[i]=​2​​len[i−1]+1​​

答案就是 dp[n]dp[n]dp[n]

代码

#include <cstdio>
#include <algorithm>
using namespace std;
const int maxd = 1e6+10;
char str[maxd];
double dp[maxd],len[maxd];
int n;
int main()
{
    // freopen("a.in","r",stdin);
    // freopen("k.out","w",stdout);
    scanf("%d%s",&n,str+1);
    for(int i=1;i<=n;i++)
    {
        if(str[i]=='x')
        {
            dp[i] = dp[i-1];
            len[i] = 0;
        }
        else if(str[i] == 'o')
        {
            dp[i] = dp[i-1] + len[i-1] * 2 + 1;
            len[i] = len[i-1] + 1;
        }
        else
        {
            dp[i] = dp[i-1] + len[i-1] + 0.5;
            len[i] = (len[i-1]+1)/2;
        }
    }
    printf("%.4lf",dp[n]);
    return 0;
}
