гҖҢCodeForces 1241EгҖҚPaint the Tree

2019-12-26 ж ‘еһӢDP

You are given a weighted tree consisting of рқ‘ӣ vertices. Recall that a tree is a connected graph without cycles. Vertices рқ‘ўрқ‘– and рқ‘Јрқ‘– are connected by an edge with weight рқ‘Өрқ‘–.

Let's define the рқ‘ҳ-coloring of the tree as an assignment of exactly рқ‘ҳ colors to each vertex, so that each color is used no more than two times. You can assume that you have infinitely many colors available. We say that an edge is saturated in the given рқ‘ҳ-coloring if its endpoints share at least one color (i.e. there exists a color that is assigned to both endpoints).

Let's also define the value of a рқ‘ҳ-coloring as the sum of weights of saturated edges.

Please calculate the maximum possible value of a рқ‘ҳ-coloring of the given tree.

You have to answer рқ‘һ independent queries.

void dfs(ll x,ll fa)
{

    for(auto p:G[x])
    {
        ll v = p.st;
        if(v == fa) continue;
        dfs(v,x);
        dp[x][0] += dp[v][1];
        dp[x][1] += dp[v][1]; //е…ҲйҖүж»Ўзҡ„еӯҗж ‘
    }
    //йҖүж»Ўзҡ„еӯҗж ‘пјҢиҝҷж ·еҪ“еүҚиҠӮзӮ№е°ұж— жі•дёҺеӯҗж ‘иҠӮзӮ№иҝһиҫ№пјҢеӣ дёәеӯҗж ‘зӮ№е·Із»Ҹж»ЎKдёӘдәҶ

    a.clear();
    for(auto p:G[x])
    {
        ll v = p.st;
        if(v == fa) continue;
        a.pb(-(dp[v][0]+ p.nd - dp[v][1]));

}
