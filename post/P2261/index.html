「CQOI2007」 余数求和

2019-05-05 数论

给出正整数 n 和 k 计算 G(n,k) = k mod 1 + k mod 2 + k mod 3 + k mod 4 + ... + k mod n 的值，其中 k mod i 表示 k 除以 i 的余数

链接🔗

[CQOI2007] 余数求和

题解

转化式子

∑i=1nk%i=∑i=1nk−i∗(k/i) \sum_{i=1}^{n} k \% i = \sum_{i=1}^{n} k - i*(k/i) i=1∑n​k%i=i=1∑n​k−i∗(k/i)

后面部分可以运用数论分块思想快速求解
当 k > n 时,上限为n

代码

#include <cstdio>
#include <algorithm>
using namespace std;
long long k,n,ans;
int main()
{
    // freopen("a.in","r",stdin);
    // freopen("k.out","w",stdout);
    scanf("%lld %lld",&n,&k);
    ans = k * n;
    long long l,r;
    for(long long i=1;i<=n;i = r + 1)
    {
        l = k/i; 
        if(l)r = min(n,k/l);
        else r = n;
        ans -= (i+r)*(r-i+1)/2 * l;
    }
    printf("%lld",ans);
    return 0;
}
