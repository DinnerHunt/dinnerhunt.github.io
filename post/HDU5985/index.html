「HDU 5985」Lucky Coins

2019-11-30 #期望概率

Bob has collected a lot of coins in different kinds. He wants to know which kind of coins is lucky. He finds out a lucky kind of coins by the following way. He tosses all the coins simultaneously, and then removes the coins that come up tails. He then tosses all the remaining coins and removes the coins that come up tails. He repeats the previous step until there is one kind of coins remaining or there are no coins remaining. If there is one kind of coins remaining, then this kind of coins is lucky. Given the number of coins and the probability that the coins come up heads after tossing for each kind, your task is to calculate the probability for each kind of coins that will be lucky.

题解

第iii种硬币,有nin_ini​枚，前kkk次以内全部反面的概率为 die[i][k]=(1−pik)nidie[i][k] = (1-p_i^k)^{n_i}die[i][k]=(1−pik​)ni​
pikp_i^kpik​ 表示这个硬币抛kkk次全都正面的概率，1−pik1-p_i^k1−pik​ 表示至少出现一次反面的概率
然后 live[i][k]=1−die[i][k]live[i][k] = 1 - die[i][k]live[i][k]=1−die[i][k] ,表示前第iii种硬币kkk次至少有一枚没有全部反面的概率
那么第iii枚硬币的答案等于

ans[i]=∑j=1200∑k=1,k!=in(live[i][j]−live[i][j+1])×die[k][j]ans[i] = \sum_{j=1}^{200} \sum_{k=1,k!=i}^n (live[i][j] - live[i][j+1]) \times die[k][j] ans[i]=j=1∑200​k=1,k!=i∑n​(live[i][j]−live[i][j+1])×die[k][j]

表示第jjj轮存活的概率加上其他所有硬币前jjj轮抛弃的概率,然后答案就是每一轮存活概率的总和

代码

#include <cstdio>
using namespace std;
const int maxd = 205;
int n,m,t;
int a[11];
double b[11];
double die[11][maxd],live[11][maxd];
double pow(double x,int y)
{
    double now = 1.0;
    while(y)
    {
        if(y%2 == 1) now *= x;
        x = x * x;
        y/=2;
    }
    return now;
}
int main()
{
//    freopen("a.in","r",stdin);
//    freopen("k.out","w",stdout);
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d",&n);
        for(int i=1;i<=n;i++) scanf("%d %lf",&a[i],&b[i]);
        if(n == 1)
        {
            printf("1.000000\n");
            continue;
        }
        for(int i=1;i<=n;i++)
        {
            for(int j=1;j<=201;j++)
            {
                double k = pow(b[i],j);
                k = 1.0 - k;

                die[i][j] = pow(k,a[i]);
                live[i][j] = 1.0 - die[i][j];
            }
        }
        for(int i=1;i<=n;i++)
        {
            double ans = 0.0;
            for(int j=1;j<=200;j++)
            {
                double now = live[i][j] - live[i][j+1];
                for(int k=1;k<=n;k++)
                {
                    if(i == k) continue;
                    now *= die[k][j];
                }
                ans += now;
            }
            printf("%.6lf",ans);
            if(i!=n) putchar(' ');
        }
        putchar(10);
    }
    return 0;
}
